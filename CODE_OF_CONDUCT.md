# Code of Conduct

This project should stay direct, technical, and respectful.

## Expected Behavior

- Focus criticism on code, design, behavior, security, and maintainability.
- Be direct without personal attacks.
- Assume contributors are trying to improve the project unless evidence says otherwise.
- Keep discussions relevant to NearShare and its local-link file sharing scope.
- Respect private security reporting rules.

## Unacceptable Behavior

- Harassment, threats, or personal abuse.
- Discriminatory language or conduct.
- Publishing private information without permission.
- Posting exploit details publicly instead of following `SECURITY.md`.
- Repeatedly derailing technical discussions.

## Enforcement

Maintainers may remove comments, close issues, restrict participation, or block users when behavior harms the project or contributors.
