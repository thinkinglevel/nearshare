# Decisions

Use this folder for ADR-style decisions once choices are made.

Suggested file names:

- `0001-android-stack.md`
- `0002-windows-stack.md`
- `0003-pairing-protocol.md`
- `0004-transfer-protocol.md`
- `0005-android-receive-mode.md`

Each decision should include:

- status;
- context;
- options considered;
- decision;
- consequences;
- follow-up work.
