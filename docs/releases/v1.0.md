# NearShare 1.0

NearShare 1.0 is the first stable release line for nearby Windows and Android file sharing.

## Artifacts

- `nearshare-android-v1.0.apk` - Android direct-install package.
- `nearshare-windows-v1.0-x64.exe` - Windows installer.
- `nearshare-v1.0-checksums.txt` - SHA-256 checksums for release packages.

## Highlights

- QR and 9-character pairing-code setup.
- Paired-device identity with signed local transfer requests.
- Android share-sheet sending.
- Windows-to-Android sending when receive mode is active.
- Manual and always-on Android receive modes.
- Private connection fallback for local route setup.
- Manual update checks through GitHub Releases.

## Notes

Same Wi-Fi is treated as reachability, not trust. Transfers still require paired-device authentication.
